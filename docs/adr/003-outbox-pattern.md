# ADR 003: outbox-pattern
## Status
Accepted
## Context
Reference platform requires production-grade architecture choices aligned with crypto brokerage and custody constraints.
## Decision
Adopt outbox-pattern as a core architectural decision for reliability, scalability, and operational maturity.
## Consequences
Positive: clear structure, extensibility, interview-ready rationale.
Negative: added complexity and operational overhead that must be mitigated with tooling and runbooks.
